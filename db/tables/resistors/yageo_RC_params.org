* Resistance value representation (for all sizes)

- Global part number uses a **2–4 digit value field**, with **R / K / M** as the decimal:
  - `97R6` → 97.6 Ω
  - `9K76` → 9.76 kΩ (9760 Ω)
  - `1M`   → 1.0 MΩ
- 1% RC0603 parts use **EIA-96 codes** (two-digit code + multiplier) for E96 values; 1% RC0805 and larger use **4-digit numeric codes** (three significant digits + exponent) for E24/E96 values.
